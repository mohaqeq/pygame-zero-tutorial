# Pygame Zero Tutorial für Pydroid 3

Ein deutschsprachiges Tutorial für Kinder, um Spiele mit Pygame Zero auf Android zu entwickeln.

## Inhaltsverzeichnis

### Teil 1: Grundlagen (keine Bilder nötig)
- `Tutorial_Teil1_Grundlagen.md` - Das Haupttutorial

**Spiele:**
- `spiele/01_grundlagen.py` - Formen zeichnen lernen
- `spiele/02_touch_eingabe.py` - Touch-Eingaben verstehen
- `spiele/03_fang_den_ball.py` - Fang den Ball
- `spiele/04_springende_box.py` - Springende Box (wie Chrome Dino)
- `spiele/05_flappy_box.py` - Flappy Box (wie Flappy Bird)
- `spiele/06_ping_pong.py` - Ping Pong gegen Computer
- `spiele/07_breakout.py` - Breakout (Steine zerstören)
- `spiele/08_button_steuerung.py` - Bildschirm-Buttons erstellen

### Teil 2: Erweitert (mit Bildern und Sound)
- `Tutorial_Teil2_Bilder_und_Sound.md` - Das erweiterte Tutorial

**Spiele:**
- `spiele_erweitert/01_flappy_bird_bilder.py` - Flappy Bird mit Bildern
- `spiele_erweitert/02_muenzen_sammeln.py` - Münzen sammeln
- `spiele_erweitert/03_space_shooter.py` - Space Shooter
- `spiele_erweitert/04_ping_pong_erweitert.py` - Ping Pong mit Effekten
- `spiele_erweitert/05_animation_beispiel.py` - Animationen erstellen

## Installation

1. **Pydroid 3** aus dem Google Play Store installieren
2. **Pydroid repository plugin** installieren
3. In Pydroid 3: Menü → Pip → `pgzero` installieren

## Ordnerstruktur für Spiele mit Bildern

```
MeinSpiel/
├── meinspiel.py
├── images/
```
